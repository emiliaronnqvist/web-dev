# JkpgCity - Web Development Project

## Setup

1. **Backend**:

   ```bash
   cd server
   npm install
   npm start
   ```

2. **Database**:

   ```bash
   docker-compose up -d
   ```

3. **Frontend**:
   The frontend is served by the backend at `http://localhost:3000`.

## Features

- List of venues (stores, restaurants, bars)
- Admin panel for CRUD operations (coming soon)
- Interactive design with Vanilla JS and CSS
